# Multi-Device UX / Fluid Continuity

## Definition
A system-focused style that ensures seamless task continuity across devices. It emphasizes consistent layouts, shared components, and adaptive behavior between mobile, tablet, and desktop.

## Intent and use
- Enable users to start on one device and finish on another without friction.
- Ideal for productivity tools, SaaS, and multi-platform products.

## Color palettes (examples)
- **System Neutral**: #F7F8FA (light), #E3E7EC (gray), #B6C0C9 (mid), #4A5560 (dark), #1A1F24 (text)
- **Calm Tech**: #F5F7FB (light), #D9E1EA (cool gray), #95A6B5 (steel), #2D3A45 (charcoal), #0F141A (text)
- **UI Base**: #FFFFFF (white), #EEF2F6 (light), #C9D2DB (mid), #52606D (dark), #1C1F24 (text)

## Design rules
- **Consistency**: same component library and tokens across breakpoints.
- **Adaptation**: responsive grids, fluid typography, and flexible spacing.
- **Continuity**: persistent navigation patterns and shared mental models.
- **Performance**: optimize for smooth transitions and low latency.

## Typography (fonts available in Penpot)
Use scalable, variable-friendly sans families.
- **Headings**: "Inter", "IBM Plex Sans", "Plus Jakarta Sans", "Manrope", "Sora"
- **Body**: same family in regular/medium
- **UI**: same family in medium/semibold

## Layout and components
- **Grid** that adapts to different viewports.
- **Navigation** that reflows but preserves structure.
- **Cards and tables** that scale with content density.

## Illustration and imagery
- Minimal, functional; avoid heavy decoration.
- Use icons that scale well across sizes.

## Accessibility
- Maintain readable sizes on small screens.
- Respect reduce motion preferences.

## Agent keywords
responsive, continuity, cross-device, adaptive, consistent, scalable, system
